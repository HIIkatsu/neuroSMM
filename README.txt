
Telegram AI Channel Bot v10

Что нового:
- полноценный редактор черновика перед публикацией
- посты руками больше не публикуются сразу
- авто-новости с trusted sources
- современный редактор: текст, картинка, кнопки, закреп, предпросмотр, отложить, опубликовать
- МСК принудительно

Установка:
1. pip install -r requirements.txt
2. Заполни .env
3. python app.py

Главные фразы:
- канал @username
- тема: массаж
- пост про массаж
- пост без картинки про массаж
- через 5 минут пост про шею
- каждые 3 дня пост про осанку
- авто-новости включить
- тема новостей: массаж
- интервал новостей: 6 часов
- источники новостей: who.int, mayoclinic.org, nih.gov
- свежая новость

Как работает редактор:
1. Пишешь тему поста
2. Бот создаёт ЧЕРНОВИК
3. Открывается меню редактора
4. Меняешь текст / картинку / кнопки / закреп / отложить
5. Нажимаешь Опубликовать
