
# NeuroSMM Mini App v2

Это уже нормальная Mini App панель, а не мёртвый шаблон.

## Что умеет
- переключение между каналами
- создание каналов и тем
- список черновиков
- создание, редактирование и удаление черновиков
- генерация черновика через твой текущий AI-слой
- публикация черновика в Telegram через токен из `.env`
- контент-план: генерация на 7/14/30/60 дней
- ручное редактирование элементов плана
- расписание слотов
- базовые настройки и статистика

## Куда класть
Распакуй **рядом** с текущими файлами бота:

- `miniapp_server.py`
- папку `miniapp/`

Пример:
```text
OlegBot/
  app.py
  db.py
  actions.py
  config.py
  miniapp_server.py
  miniapp/
```

## Запуск локально
Установи зависимости, если их ещё нет:
```bash
pip install fastapi uvicorn
```

Запусти сервер из папки бота:
```bash
uvicorn miniapp_server:app --host 0.0.0.0 --port 8000
```

Открой в браузере:
```text
http://127.0.0.1:8000
```

## Как открыть в Telegram
Для реального Mini App нужен HTTPS.
Пока бот у тебя локально, самый простой путь — поднять туннель, например cloudflared или ngrok, и дать Telegram HTTPS URL.

Потом в боте даёшь кнопку `web_app=WebAppInfo(url="https://твой-домен")`.

## Важно
- Для публикации и AI-генерации нужны рабочие `.env`, `BOT_TOKEN` и `OPENROUTER_API_KEY`.
- Загрузка файла с телефона как картинки пока **не добавлена** специально, чтобы не ломать твой текущий бот.
  Сейчас можно указывать прямой URL картинки или уже существующий Telegram `file_id`.
- Сервер использует твою текущую БД и функции из `db.py`, поэтому это уже не демка на localStorage.

## Что лучше делать дальше
Следующий шаг уже логичный:
1. перенести бота на VPS
2. повесить домен / HTTPS
3. добавить кнопку открытия Mini App в самого бота
4. потом расширить редактор: загрузка изображений, предпросмотр, drag-drop календарь, авто-новости, аналитика
